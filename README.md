# spring boot + dubbo demo

### install zookeeper and start zookeeper
[click download](http://www-eu.apache.org/dist/zookeeper/stable/zookeeper-3.4.12.tar.gz)
```bash
# start zookeeper use default conf
# cd zookeeper dir
cd bin
./zkServer start
```
